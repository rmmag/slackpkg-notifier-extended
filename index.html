Welcome to Slackware Notifier - Extended.

This is modified and extended version of https://github.com/guax/slackpkg-notifier

Modifications:

  • Fixed to working with new slackpkg version
  • Added Hide icon option in tray
  • Notifier automaticly checking for updates available for your system
  • If you have installed packages coming from no-official repository, notifier mark them and will not be cheking enymore
  • Right click on 'U' icon in tray makes run updating and upgrading system
